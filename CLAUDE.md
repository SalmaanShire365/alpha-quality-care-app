# Project: Alpha Quality Care1 Website

A modern healthcare services website rebuilt using Next.js to provide clear service information, contact capabilities, and a professional online presence.

## Tech Stack
- Next.js (App Router)
- React
- Tailwind CSS
- Vercel (deployment)

## Project Structure
- app/ → pages and routing
- components/ → reusable UI components
- public/ → images and assets

## Commands
- npm run dev → start dev server
- npm run build → production build
- npm run lint → lint code

## Development Workflow
- Build UI in reusable components
- Keep pages clean and minimal
- Use Tailwind for all styling
- Prioritize responsiveness

## Key Features
- Homepage with services overview
- About page
- Services page
- Contact form

## Guidelines
- Prefer simple, readable code over complex abstractions
- Reuse components wherever possible
- Keep components under 200 lines
